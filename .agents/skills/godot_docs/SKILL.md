# Instructions

When writing or modifying Godot Engine code (`.gd` scripts, `.tscn` scenes, etc.), you MUST always reference the documents located in the `G:\project\gun_ship\docs` folder for guidance, best practices, and project-specific conventions.

- Use your file reading tools (like `list_dir`, `view_file`, or `grep_search`) to check the contents of `G:\project\gun_ship\docs` before making architectural decisions or writing new Godot code.
